LeetCode → GitHub Syncer

Privacy Policy

1. Introduction

This extension ("LeetCode → GitHub Syncer") is an open-source, client-side-only tool designed to synchronize your LeetCode submissions directly to your GitHub repository.

2. Data Collection and Usage

  • GitHub Personal Access Token (PAT): Your PAT is stored locally in your browser to authenticate with the GitHub API. If you select the "Remember token" option, it is saved to your browser's persistent local storage. If unchecked, it is held in memory (session storage) and cleared when the browser closes. It is used exclusively to push code to the specific repository you define.
  • LeetCode Session Cookie: The extension requires the cookies permission to read the csrftoken cookie scoped specifically to leetcode.com. This cookie is used strictly to authenticate GraphQL requests to LeetCode's API to fetch your submission details. No cookie value is ever stored on disk, logged to the console, or transmitted to any third party.

3. No Third-Party Tracking

This extension respects your privacy:

  • We do not collect analytics or telemetry.
  • We do not use remote servers or intermediate APIs. Your data travels directly between your browser, LeetCode, and GitHub.
  • We do not sell or share your data.

4. Data Deletion

To delete all data associated with this extension, simply uninstall it from your browser or click "Clear" in the extension settings. Since no data leaves your browser, this permanently removes everything.

5. Contact

If you have any questions about this privacy policy, please open an issue on our GitHub repository.
