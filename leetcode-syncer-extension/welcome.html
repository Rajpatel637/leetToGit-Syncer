Welcome to LeetCode → GitHub Syncer! 🎉

You're almost ready to automate your LeetCode portfolio. This extension runs locally in your browser and automatically pushes your Accepted solutions to a GitHub repository of your choice.

  1. Create a new repository on GitHub (e.g., leetcode-solutions).
  2. Generate a fine-grained Personal Access Token scoped to that repository with Contents: Read & write permissions.
  3. Open the extension settings and paste your token and repository name.
  4. Solve a problem on LeetCode. It will sync instantly!
